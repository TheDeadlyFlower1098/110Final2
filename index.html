Grocery Manager

Make a grocery list manager. It must have the following features -

VSCode, JavaScript
  • All items will be displayed in the page at all time. Don't forget to strikethrough the already-purchased products.
  • How to strikethrough a text in HTML, you ask? Google it up. Make a habit to Google stuff you don't know.
  • Clicking on any unpurchased item will cross it out. Clicking on it again will undo it.
  • Add items.
  • View only items left to purchase.
  • View all the purchased items.

Quiz App

Make a Quiz App with the following features -

VSCode, Node.js, HTML, CSS, JavaScript
  • Store the questions and answers in an array of objects in your JS file.
  • The options each are radio buttons. You can Google radio buttons in HTML for more information. The best site for these materials is probably w3school website
  • Don't let the user submit unless an option is selected.
  • When all the questions are answered, the score is displayed and a button to let the user reload and start from beginning. Google up how to reload a page in JS if needed or come up with your own solution.
  • Design doesn't matter
